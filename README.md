# Emotion Detector

## Project Overview

Emotion Detector is a Python-based AI web application that analyzes text and detects emotions such as:

- Joy
- Anger
- Sadness
- Fear
- Disgust

The project was developed using Flask and Python as part of the IBM Skills Network final project.

---

## Features

- Emotion analysis from user text
- Flask web application deployment
- Error handling for blank inputs
- Unit testing using unittest
- Static code analysis using pylint
- Modular package structure

---

## Technologies Used

- Python
- Flask
- Requests
- Pylint
- GitHub
